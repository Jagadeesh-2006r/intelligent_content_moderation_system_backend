# SentriAI Backend

## Overview

The SentriAI Backend manages API requests, processes user uploads, communicates with the AI engine, and integrates with UiPath Maestro Case for intelligent content moderation.

## Features

* REST API
* Content processing
* AI engine integration
* UiPath Maestro Case integration
* Secure request handling
* Real-time moderation workflow

## Tech Stack

* Python (Django)
* FastAPI
* REST API

## Run Locally

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

## Repository

Backend service for the **SentriAI – Intelligent Content Moderation System**.
